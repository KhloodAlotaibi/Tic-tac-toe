# Tic-tac-toe

Tic Tac Toe is a famous puzzle game also known as "Noughts and Crosses" or "X and O".

App Features:
- One and two player mode
- Show scores

لعبة اكس او الشهيره
- خاصية اللعب الفردي 
- خاصية اللعب ضد صديق 
- جمع النقاط
